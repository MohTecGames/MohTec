DESCRIPTION:
A single-file, responsive personal portfolio website 
designed with a "Midnight Neon" aesthetic. It features 
an animated night sky background (moon, stars, clouds), 
glassmorphism floating cards, and a technical skill matrix.

FEATURES:
- Single HTML file (contains all HTML, CSS, and JavaScript).
- Interactive "Expandable" experience cards.
- Animated CSS star-field and drifting clouds.
- Neon text effects and dark-mode glassmorphism.
- Fully responsive for mobile and desktop.
 
TECHNICAL STACK:
- HTML5
- CSS3 (Animations, Flexbox, Grid)
- Vanilla JavaScript (No external libraries required)
